Prédiction de données maritimes
-
(IA, Big Data & Visualisation Web)

Prédiction de données maritimes
2024 -2025

Ce projet de fin d’année alliait Intelligence Artificielle, Big Data et Développement Web. L’objectif était de concevoir une solution capable d'analyser et de prédire le comportement de navires maritimes à partir de données massives.

Nous avons assuré la préparation et le nettoyage des données, développé des modèles de prédiction (à l’aide de Scikit-learn et XGBoost), puis conçu une interface web interactive pour visualiser les résultats.

Ce projet m’a permis de renforcer mes compétences en traitement de données, modélisation et visualisation, tout en découvrant le potentiel de l’IA pour l'analyse prédictive et la prise de décision.

Outils : Scikit-learn, XGBoost, Pandas, NumPy, Jupyter Notebook, HTML/CSS/JavaScript, PHP, MySQL, Plotly, Mapbox, Git

Robot Raspberry Pi
2023 -2024

Ce projet a été réalisé dans le cadre d'une SAé programmation lors du troisième semestre de mon BUT R&T.
